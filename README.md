# Aurora Qvick Core

Aurora Qvick Core is an AI-powered digital companion project developed by Qvick Games.

The long-term goal is to build a natural AI companion that combines:

- Conversational AI
- Long-term memory
- Voice interaction
- Cinematic user interface
- Immersive environments
- Game development assistance

Aurora is designed to evolve beyond a traditional chatbot into a persistent digital companion.

## Technologies

- React
- TypeScript
- Firebase
- Google AI Studio
- Vercel
